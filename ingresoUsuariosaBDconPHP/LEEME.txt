Step 3: Test PHP Installation

Open Command Prompt (CMD).

Run the following command to verify PHP is installed: 
php -v 
You should see the PHP version details if everything is set up correctly.

Step 4: Configure php.ini 

Navigate to C:\php.

Copy php.ini-development and rename it to php.ini.

Open php.ini in a text editor and enable required extensions by uncommenting lines (remove ;), e.g.:

extension=curl
extension=mbstring
extension=pdo_mysql

Save the file.

Step 5: Run PHP Scripts

To execute a PHP script:

Navigate to the script's directory in CMD:

cd path\to\your\script
